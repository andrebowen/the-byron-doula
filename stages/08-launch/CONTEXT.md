# Stage 08 — Launch

## Purpose
Make the site live.

### Activities
- Domain setup
- Hosting configuration
- Analytics installation
- Search engine indexing

### Deliverables
- Live website

## Inputs
- QA-approved build (from Stage 07-testing-qa)

## Outputs
- Live website at the production domain
- Analytics and tracking live
- Site submitted to search engines

## Process
1. Configure production hosting / domain (DNS, SSL).
2. Deploy the final build.
3. Install analytics, search console, etc.
4. Submit sitemap for indexing.
5. Announce / update any marketing assets.
6. Update `../../MEMORY.md` — mark complete and move to maintenance.

## Checkpoint
Site is live and verifiable. Analytics receiving data.
